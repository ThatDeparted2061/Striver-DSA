## 🎓 How to Track Progress

### Using Markdown:
Replace `☐` with:
- `✅` - Solved & understood
- `🔄` - In progress
- `⏳` - Todo/Later
- `⭐` - Mark as favourite

### Adding Notes:
Examples of good notes:
- `Two-pointer approach, O(n) time`
- `HashMap for O(1) lookup`
- `Took 25 min first try`
- `Revisit: tricky edge case`

- `Use sliding window pattern`
