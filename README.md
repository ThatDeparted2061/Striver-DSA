## 🎯 How to Use

3. **Track your progress**:

## 🎓 How to Track Progress

### Using Markdown Checkboxes:
Replace `☐` with:
- `✅` - Solved & understood
- `🔄` - In progress
- `⏳` - Todo/Later
- `⭐` - Mark as favourite

### Adding Notes:
Examples of good notes:
- `Two-pointer approach, O(n) time`
- `HashMap for O(1) lookup`
- `Took 25 min first try`
- `Revisit: tricky edge case`
- `Similar to Problem #5`
- `Use sliding window pattern`

### Progress Summary (Create Your Own):
Track weekly progress by adding a section:
```markdown
## 📊 My Progress
```
